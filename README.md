## Requirements
- docker
- docker-compose

## How to run locally
Clone the repository:
```bash
git clone git@github.com:Alzex/be-labs.git
```

Go to the project directory:
```bash
cd be-labs
```

Run the project with `docker-compose`:
```bash
docker-compose up --build
```
